# ⚽ NARRYBET

Casa de apostas pessoal + acompanhamento de gameplay do **PES 13 Rumo ao Estrelato**.

## Funcionalidades
- 🎮 Registro de partidas com eventos ao vivo
- 🥜 Sistema de Amendoins com bônus por tipo de gol
- 🎲 Apostas com odds inteligentes por jogador
- ⚔️ Histórico de duelos por adversário
- 📈 Gráfico de forma e calendário visual
- 🏁 Sistema de temporadas com arquivo histórico
- 🎯 Objetivos semanais e mensais

## Acesso
